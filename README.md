# Konachan Prettify

Chrome extension

Adjust Konachan website UI for better experience

### Features
- Support yande.re
- Bigger preview images
- Blank space at right side removed
- Image size info bar removed
- Konachan.com ads removed (→_→)
- Both .com and .net are supported
